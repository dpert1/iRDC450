Equation 32: Cube Root Scaling (Multiple Parameters)

Equations:

\[d = d_1 \times W^{1/3}\]
\[t = t_1 \times W^{1/3}\]
\[h = h_1 \times W^{1/3}\]

Variables:

  • \(d\): Blast wave arrival distance [m]
  • \(t\): Blast wave arrival time [s]
  • \(h\): Blast wave height [m]
  • \(d_1, t_1, h_1\): Reference values for 1 kt yield
  • \(W\): Weapon yield [kt]

Description:

Cube root scaling applies to various blast wave parameters including distance, time, and height, all scaling with W1/3.
